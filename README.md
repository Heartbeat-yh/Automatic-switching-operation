# 多人协同运营自动化系统

> 面向企业复杂运营场景的智能协同自动化平台。  
> 通过「前端控制台 + 后端 API + 多 Agent 调度 + 任务对流 + 飞书通知站位 + SQLite 数据库」，实现从自然语言任务输入、任务拆解、多人协作、Agent 执行、结果复核、通知触达、审计沉淀的完整闭环。

---

## 一、项目简介

多人协同运营自动化系统，是一个面向企业内部运营、客服、财务、风控、售后、交付等多角色协作场景的智能化任务处理平台。

传统运营系统通常只能做到：

- 人工创建任务；
- 人工分配负责人；
- 人工沟通进度；
- 人工整理结论；
- 人工通知相关人员。

但在真实业务中，很多运营问题并不是一个简单待办，而是一个复杂链路。例如：

```text
用户反馈订单异常
  ↓
客服初步确认
  ↓
运营查询订单
  ↓
财务核对支付流水
  ↓
技术排查接口日志
  ↓
判断是否需要退款 / 补偿 / 重新分账
  ↓
通知负责人处理
  ↓
汇总结果并回复用户
